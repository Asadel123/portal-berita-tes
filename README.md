# portal-berita-tes

Ini adalah proyek portal berita yang dibangun menggunakan Laravel.

## Fitur
- Halaman utama berita.
- CRUD untuk berita.

## Instalasi
1. Clone repositori ini.
2. Jalankan `composer install`.
3. Buat file `.env` dan konfigurasi database.
